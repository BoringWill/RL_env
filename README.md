# Slime Volleyball - 双人对战版 (Python)

这是一个基于 Pygame 开发的经典史莱姆排球游戏。本项目采用模块化设计，重点优化了物理碰撞体验和双人同屏操作。


## 📂 文件结构

| 文件名 | 描述 |
| :--- | :--- |
| `main.py` | **程序入口**。负责键盘事件监听、得分逻辑判定、屏幕刷新及渲染。 |
| `entities.py` | **核心逻辑**。定义了 `Entity` 类和 `SlimeBall` 类，包含物理位移及复杂的碰撞算法。 |
| `constants.py` | **参数配置**。包含所有物理常量（重力、速度、球半径、网高度、颜色等）。 |

## 🚀 快速开始

1.  **安装环境**：
    确保已安装 Python 环境，并安装 `pygame` 库：
    ```bash
    pip install pygame
    ```

2.  **运行项目**：
    在文件夹目录下运行：
    ```bash
    python main.py
    ```

## 🕹️ 操作说明

| 玩家 | 角色 | 移动 (左/右) | 跳跃 |
| :--- | :--- | :--- | :--- |
| **Player 1** | 左侧红色史莱姆 | `A` / `D` | `W` |  (开始请按空格, 记得按空格)
| **Player 2** | 右侧蓝色史莱姆 | `左方向键` / `右方向键` | `上方向键` |

> **重要提示**：运行游戏时，请务必将电脑输入法切换至 **英文状态**，并且打开**大写锁定**否则 P1 的 `WAD` 键将无法正常工作。

## 🛠️ 自定义设置 (constants.py)

你可以通过修改 `constants.py` 来改变游戏的难度和手感：

* **弹跳高度**：减小 `GRAVITY` 的值（如调至 `0.3`）可以让球弹得更高、更慢。
* **球体大小**：修改 `BALL_RADIUS` 的值。
* **网格挑战**：修改 `NET_HEIGHT` 来增加或减少过网难度。
* **球速上限**：修改 `BALL_MAX_SPEED` 来控制比赛节奏。
